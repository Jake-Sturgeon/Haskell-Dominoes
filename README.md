# Dominoes

A game of Dominoes with several player types.

## Installation

In this example I used GHCI to run and interpretate the program 

## Usage

To load the environment
```bash
ghci
```
To load the code
```bash
:l DomsMatch
```
To run the code: 
```bash
domsMatch bestPlayer bestPlayer [NUMBER OF GAMES] [SEED]
```
